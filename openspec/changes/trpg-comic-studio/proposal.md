## Why

为 TRPG（桌上角色扮演游戏）跑团 Replay 创作者提供一个集**动态漫画编辑、动画时间线、AI 素材生成、视频导出**于一体的桌面应用。现有方案需要剪辑软件 + AI 生图工具 + 排版工具分散操作，工作流割裂且效率低下。本应用将所有环节整合到一个创作环境中，个人使用，追求可扩展性以支持持续迭代。

## What Changes

- 新增**三层图层制作面**（漫画框层 / 人物层 / 对话气泡层），支持图片的插入、变换与混合模式
- 新增**时间线动画系统**：每个图像独立轨道，支持拖拽效果块，效果可叠加组合产生复合动画
- 新增**效果插件架构**：内置平滑移动、淡化、缩放、果冻/呼吸、材质覆盖、帧动画、UT风格文字动画，支持后期扩展新效果
- 新增**可变形对话气泡**：Mesh 网格形变 + 内部文本自动重排
- 新增**Log 分屏展示**：解析海豹骰服务器的 docx 格式 Log，保留富文本格式
- 新增**素材库**：树形文件管理器，支持多层分类、动画预设保存/复用
- 新增**AI 素材生成**：集成 ComfyUI 本地引擎（主）与云端 API（备），支持 LoRA 微调保角色一致性
- 新增**预览与导出**：实时播放动态漫画，一键导出 MP4 视频
- 新增**项目管理系统**：文件夹式项目结构、无限撤销/重做、自动保存

## Capabilities

### New Capabilities

- `project-management`: 项目文件的创建、打开、保存、自动保存，撤销/重做历史栈，文件夹式项目结构
- `canvas-editor`: 三层图层（底层漫画框 / 中层人物 / 顶层气泡）的 PixiJS 渲染引擎，图片的插入、选中、移动、缩放、旋转等基础变换操作
- `timeline-system`: 类必剪风格的多轨时间线编辑器，每个图像 Item 独立轨道，效果块的创建、拖拽、时长调整、吸附对齐，播放头控制
- `animation-effects`: 效果插件系统，内置平滑移动、淡化、缩放、果冻/呼吸、材质覆盖、帧动画、UT 文字动画，支持效果叠加组合
- `bubble-editor`: 可变形的对话气泡组件，9点 Mesh 网格拖拽形变，内部文本根据气泡边界自动重排
- `log-viewer`: 制作面右侧 Log 分屏，解析 docx 格式的跑团记录，保留原文本字体颜色与格式
- `asset-library`: 右侧树形文件管理器，支持多级文件夹分类、图片/音乐/音效管理、动画预设保存与复用
- `ai-generation`: ComfyUI 本地引擎桥接 + 云端 API 备选，LoRA 自动训练流程，IP-Adapter 参考图生图，多智能体画风管理
- `preview-export`: 内置播放逻辑（图层顺序、静默保持），FFmpeg 帧序列合成导出 MP4

## Impact

- **技术栈**：TypeScript + Electron + React + PixiJS v8 + Zustand + Vite
- **外部依赖**：ComfyUI（Python 进程，本地运行）、FFmpeg（视频编码）、mammoth（docx 解析）
- **存储**：文件夹式项目结构，JSON 序列化 + Schema 版本号，素材外部路径引用
- **平台**：Windows 优先（个人使用），macOS 后续适配
- **分发**：不涉及分发，本地开发运行
