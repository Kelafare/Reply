# 跑团Replay动态漫画桌面应用产品说明报告

## 2. 功能清单

### 2.1 制作面模块

制作面是核心创作区域，采用三层图层架构，支持精细化的时间线控制。

功能子项

详细描述

**三层图层架构**

**底层**：分镜漫画框层，支持边框设定及背景图填充；**中层**：人物图像层；**顶层**：对话气泡层。所有层级均支持独立图片插入与混合模式。

**时间线动画系统**

采用类必剪的时间线交互，支持拖拽“效果块”。每一个图像都会有一个时间线用来设置各个动画效果的起止时间，仅带结束时间的效果可生成块并拖拽，无结束时间效果为该图像的全局/持续属性。

**平滑移动效果**

支持起点与终点坐标设定，速度曲线可调节，具有明确结束时间。

**淡化动画效果**

支持透明度渐变与速度调节，可设置动画结束后是否自动还原至未淡化状态。

**缩放动画效果**

支持放大/缩小平滑过渡，大小目标值与速度可调节，具有明确结束时间。

**果冻/呼吸效果**

类似骨骼绑定，通过轻微起伏营造呼吸感或晃动效果。骨骼设置与晃动幅度可调，**无结束时间**（持续生效）。

**材质覆盖效果**

保持图片轮廓不变，叠加材质或调整体色。可设置持续时间及恢复正常的时刻，持续时间即为结束时间。

**图片帧动画设置**

选中图像时底部显示已有帧列表，支持插入新帧。可设置单帧时长、是否循环，若需结束时间则必须设定。

**气泡网格扭曲**

对话气泡四边中点设有节点，拖拽可扭曲形状以适配漫画框。**内部字体随网格同步形变**（如边缘压扁，字体对应端点亦压扁）。

**UT风格文字动画**

聊天气泡专属：支持短音频绑定，实现“一字一音”效果；可选“一次性全显”或“逐字显示”模式。

**Log展示与排版**

开启后，制作面右侧预留 **30%** 空间展示文本Log。支持复制粘贴并**保留原文本字体颜色及格式**，文本区与漫画区并存。

**基础编辑功能**

支持草稿自动/手动保存；支持 Ctrl+Z 无限级撤销操作。

**内置播放逻辑**

按图层顺序（底->中->顶）依次播放图像效果，且可以设置是每层都按照安放图像的顺序一个一个显现还是一下子一层全都显现。当前图像时间线结束后显现下一张；支持设置所有图像显示后的**静默保持时长**，随后自动进入下一页。

### 2.2 素材面模块

素材面负责AI生成与资产管理，支持多智能体协作。

**• 人设/画风投喂**：支持上传参考图与搭配提示词，指导大模型生成特定角色或风格图片。

**• 多智能体管理**：允许新建多个智能体，分别负责不同画风的图片生成。画风由投喂的图片决定

**• 全品类生成**：除角色外，支持生成背景图、道具、物品等素材。

**• 生成结果管理**：生成的图片可直接保存至素材库复用，支持批量管理与筛选。

### 2.3 素材库模块

素材库以**右侧树形文件管理器**形式常驻，支持外拉扩展。

**• 分类管理**：支持新建多级文件夹，对图像、音乐、预设进行逻辑分类。

**• 动画预设保存**：用户可将设置好动画效果的图像保存为“预设图像”。再次调用时，**自动继承所有时间线、效果块及参数**，极大提升批量制作效率。

**• 多媒体支持**：除图片外，支持存放背景音乐、音效文件，方便在制作面直接拖入时间线。

### 2.4 预览与导出模块

**• 实时预览**：支持在当前页面直接播放动态漫画，检查动画衔接与Log排版。

**• 视频生成**：一键将当前工程导出为 MP4 视频，支持分辨率与帧率设置。

## 3. 技术实现方式

### 3.1 桌面应用架构

**• 核心语言**：**TypeScript**，确保大型项目的类型安全与可维护性。

**• 桌面框架**：

**◦ 首选方案：Electron**。利用 Node.js 生态处理本地文件读写、FFmpeg 视频合成及素材库管理，Chromium 内核保证复杂 Canvas 渲染的兼容性。

**◦ 备选方案：Tauri**。若对包体积与内存有极致要求，采用 Rust 后端 + Web 前端，通过 IPC 调用本地生图服务。

### 3.2 渲染与动画引擎

**• 2D渲染核心**：**PixiJS** (WebGL)。负责三层图层的高性能合成、Mesh 网格形变（气泡扭曲）、骨骼动画（果冻效果）及硬件加速。

**• 时间线逻辑**：基于 **GSAP** 或自定义 Timeline 引擎，实现效果块的拖拽、吸附、速度曲线编辑及帧动画逻辑。

**• 交互层**：**Fabric.js** 或 **Konva.js**，处理画布上对象的选中、节点拖拽、气泡形变控制点等复杂交互。

### 3.3 AI 生图与角色一致性方案

**• 模型基座**：**Stable Diffusion (SDXL / SD 1.5)**，兼顾画质与显存占用。

**• 接入方式**：

**◦ 云端API**：集成 **阿里云 DashScope (万相)** 或 **Replicate**，支持 FaceChain 免训练写真生成，降低用户本地硬件门槛。

**◦ 本地引擎**：后台静默运行 **ComfyUI**，通过 JSON API 调用工作流，支持离线生成与隐私保护。

**• 角色一致性技术栈**：

**1. LoRA 微调**：用户上传 5-15 张角色图 -> 后台自动触发 Kohya\_ss 训练 -> 生成轻量级 LoRA 权重 -> 存入素材库。生图时自动加载，确保面部/服装高度一致。

**2. IP-Adapter + ControlNet**：作为“免训练”补充方案。上传单张人设图作为 Reference，模型在生成新动作/背景时自动“抄”特征，实现秒级一致性生成。

性能测试报告、用户操作手册、正式安装包
